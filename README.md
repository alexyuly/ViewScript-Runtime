## Start

1. Install ViewScript

```
npm install viewscript --global
```

2. Create a new project

```
viewscript project HelloWorld
```

3. Run your app

```
cd HelloWorld && npm start
```

## Hello, World!

```
View HelloWorld {
   <p>
      content = "Hello, world!"
}
```

HelloWorld is a view, with one paragraph element, with content of "Hello, world!"
